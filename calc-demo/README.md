## calc sample

简单计算公式

### 使用方法

- Python3

`python3 main.py [a] [b]`

```bash
# ronething @ ashings-macbook-pro in ~/Documents/calc on git:master x [13:13:50]
$ python3 main.py 2.5 60
(a * 10^6) / (14.3 * 1000 * b^2) =  0.049
1 - sqrt(1 - 2 * c) =  0.050
1 - 0.5 * d =  0.975
(a * 10^6) / (360 * e * b) =  119


```
